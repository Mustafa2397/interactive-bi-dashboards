# Business Intelligence Portfolio ◈

مجموعة من لوحات البيانات التفاعلية المصممة لتحويل البيانات الخام إلى رؤى استراتيجية تدعم اتخاذ القرار، مع التركيز على دقة البيانات وجودة العرض البصري.

## 📊 Featured Project: Risk & Sustainability Analysis
مشروع متكامل لتحليل مخاطر الاستدامة وسيناريوهات التهديد البيئي باستخدام Power BI، يهدف إلى تقديم رؤية واضحة لمتخذي القرار حول مستويات الخطر التاريخية والمستقبلية.

### **🛠 القدرات التقنية المنفذة:**
* **Data Cleaning (ETL):** فحص جودة البيانات وضمان خلوها من الأخطاء (Data Profiling) باستخدام Power Query لضمان دقة النتائج بنسبة 100%.
* **Advanced Risk Modeling:** تحليل "أيام الخطر" (Danger Days) والارتباط بمعدلات الأمطار وسيناريوهات التهديد المختلفة.
* **Business Insights:** تتبع اتجاهات المشاريع وتحليل الأنماط الزمنية لتحسين التنبؤات المستقبلية.
