# Business Intelligence Portfolio ◈

مجموعة من لوحات البيانات التفاعلية المصممة لتحويل البيانات الخام إلى رؤى استراتيجية، مع التركيز على دقة البيانات وجودة العرض البصري.

## 📊 Featured Project: Risk & Sustainability Analysis
هذا المشروع يحلل مخاطر الاستدامة وسيناريوهات التهديد البيئي باستخدام Power BI.

### **المميزات التقنية:**
* **Data Cleaning (ETL):** تم فحص جودة البيانات وضمان خلوها من الأخطاء باستخدام Power Query.
* **Risk Modeling:** تحليل "أيام الخطر" (Danger Days) والارتباط بمعدلات الأمطار.
* **Business Insights:** تتبع دورة حياة المشاريع وحالاتها (ناجحة/فاشلة) تاريخياً.
